\subsection{Description détaillée}
La classe \hyperlink{class_point}{Point} représente les points ou sommets(composés de 3 coordonnées) des triangles \hyperlink{class_face}{Face} composant l'objet \hyperlink{class_mesh}{Mesh}. Chaque sommet du triangle a donc trois points dans l'espace (coordonnées). \par
Il s'agit donc ici de récupérer les coordonnées se trouvant dans le fichier .O\-F\-F . Ces coordonnées sont alors placées respectueusement dans les attributs point\-\_\-one, point\-\_\-two, point\-\_\-three de type {\bfseries double} \par
