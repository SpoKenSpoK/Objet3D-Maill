\subsection{Description détaillée}
La classe \hyperlink{class_mesh}{Mesh} représente l'objet maillé 3\-D. Ce dernier contient un certain nombre de points et faces entiers. \par
Nous utilisons donc ici les types {\bfseries unsigned} {\bfseries int} pour les attributs.
