Description détaillée

La classe Point représente les points ou sommets(composés de coordonnées) des triangles Face composant l'objet Mesh. Chaque sommet du triangle a donc trois points dans l'espace(coordonnées).
Il s'agit donc ici de récupérer les coordonnées se trouvant dans le fichier .OFF . Ces coordonnées sont alors placées respectueusement dans les attributs point_one, point_two, point_three de type double
Par la suite il nous sera donc possible d'aller chercher une coordonnée suivant la ligne de lecture qui nous sera indiquée (cf. main.cpp & explication de Face).
Une méthode de calcul est aussi présente, celle-ci récupère la différence entre deux coordonnées \((x2-x1)^2\).
Il faudra alors la répeter trois fois pour obtenir la longueur d'un segment dans l'espace (en y ajoutant la racine carré). Soit :
\(\sqrt{(x2-x1)^2+(y2-y1)^2+(z2-z1)^2}\) (cf. main.cpp)
